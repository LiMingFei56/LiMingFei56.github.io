#### 1. HTTP事务

    HTTP服务器: 使用HTTP协议共享资源的Web服务器
    HTTP客户端: 使用HTTP协议请求存储在HTTP服务器上的资源
    HTTP事务: 由HTTP客户端的HTTP请求，HTTP服务器的HTTP响应构成
