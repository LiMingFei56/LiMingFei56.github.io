Genericity

### Templates

模板是泛型编程的基础，它涉及以独立于任何特定类型的方式编写代码

#### Function Template

	template <class type> ret-type func-name(parameter list) {
	   // body of function
	}

#### Class Template

	template <class type> class class-name {
	   .
	   .
	   .
	}
