### 1. 以"(", "[", "/", "+", "-" 开始, 那么它极有可以和前一条语句合在一起解析

    var y = x + f
    (a + b).toString()

    var y = x + f(a + b).toString();

### 2. return, break, continue后紧跟着换行, 则会在换行处填补分号

    return
    true;

    return; true;

### 3. "++", "--" 需要跟表达式在一行, 不然会与后面的表达式一起解析

    x
    ++
    y

    x; ++y
